Class: AudioEngine

AudioEngine(gameEngine, filename)

Creates an instance of Csound and compiles a file called AudioEngine.csd by default. Custom .csd files can also be loaded.

Constructor

new AudioEngine(gameEngine, filename)

Parameters:
Name Type Default Description
gameEngine string Set the game engine being used, i.e., 'Babylonjs' or 'Phaser3'
filename string AudioEngine.csd the .csd file to compile.
Source:

createAudioSource(source, listener, filename)

Creates an audio source whose amplitude is determined by its distance to the listener. The name of the source is used to create unique channels that the audio engine can later send data to.
